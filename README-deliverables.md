## Overview

This starter kit provides a practical first layer of AI governance documentation for EU teams already using AI in everyday business operations. It helps teams move from undocumented AI usage to a credible, editable governance baseline.

**What this kit is:**
- A documentation starting point, not a compliance certification
- Editable templates adaptable to your team's context
- A practical foundation for ongoing governance

**What this kit is NOT:**
- Legal advice
- Guaranteed compliance with the EU AI Act
- A compliance certification
- A bespoke audit
- A full implementation project

## Usage Recommendations

### First Week

1. **Read through all templates** to understand the governance scope
2. **Customize the AI Usage Policy** to your team's context
3. **Populate the AI Systems Register** with current tools and use cases
4. **Assign owners** for each document area
5. **Conduct a 60-minute first pass** to get the basics documented

### Ongoing

- Update the register when new AI tools are introduced
- Log incidents as they occur
- Review and update policy quarterly
- Conduct annual reviews of all documentation
